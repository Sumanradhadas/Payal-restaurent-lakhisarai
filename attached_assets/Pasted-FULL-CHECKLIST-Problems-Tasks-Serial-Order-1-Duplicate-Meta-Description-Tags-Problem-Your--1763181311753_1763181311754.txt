FULL CHECKLIST — Problems + Tasks (Serial Order)
1️⃣ Duplicate Meta Description Tags
Problem:

Your site has 2 <meta name="description"> tags.

Google hates duplicate description tags.

Task:

Remove one description tag.

Keep only ONE description:

<meta name="description" content="Payal Restaurant in Purani Bazar, Lakhisarai — homely vegetarian thalis, North & South Indian favourites and sweets. Order via Zomato/Swiggy or call +91 63462 33115.">

2️⃣ Duplicate Canonical Tags
Problem:

Page contains 2 canonical links.

Google must see only ONE canonical.

Task:

Remove one canonical tag.

Keep only:

<link rel="canonical" href="https://payal-restaurent-lakhisarai.onrender.com"/>

3️⃣ 404 Page Returns 200 OK (MAJOR CRITICAL ISSUE)
Problem:

Missing pages (example: /test123) return 200 OK instead of 404 Not Found.

Google treats these as soft 404 → indexing problem.

Task:

Fix your 404 routing:

If using React/Vite SPA

Create public/_redirects file:

/*    /index.html   200
/404   /404.html    404


Add a real 404.html file.

4️⃣ No OpenGraph Tags
Problem:

Missing OG tags → bad social preview, weaker SEO.

Task:

Add OG tags to <head>:

<meta property="og:title" content="Payal Restaurant, Lakhisarai">
<meta property="og:description" content="Homely vegetarian thalis & local favourites.">
<meta property="og:url" content="https://payal-restaurent-lakhisarai.onrender.com">
<meta property="og:image" content="/og-image.jpg">

5️⃣ Text-to-Code Ratio is Too Low (6%)

Reduce unnecessary divs/animations.

9️⃣ Missing Structured Data (Schema)
Problem:

No schema for LocalBusiness.

Google prefers structured data for restaurants.

Task:

Add schema JSON-LD:

{
  "@context": "https://schema.org",
  "@type": "Restaurant",
  "name": "Payal Restaurant",
  "address": "Purani Bazar, Lakhisarai",
  "servesCuisine": ["Indian", "Vegetarian"],
  "telephone": "+916346233115",
  "url": "https://payal-restaurent-lakhisarai.onrender.com"
}
</script>
